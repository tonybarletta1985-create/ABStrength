# App Icons

Place your app icons here. Required sizes for PWA:
- icon-192.png (192×192)
- icon-512.png (512×512)

For Apple devices (optional but recommended):
- icon-152.png (152×152)
- icon-167.png (167×167)
- icon-180.png (180×180)

You can generate icons for free at: https://realfavicongenerator.net
Or use any image and resize it with: https://squoosh.app

A simple "AS" or dumbbell icon on a blue background works well.
